# ListenShelf 0.1.0 Alpha 1

This is the first public Windows preview of ListenShelf. It is intentionally small: a working offline M4B player and the beginning of the local library experience.

## Included

- Import one or more local M4B audiobooks into a linked or ListenShelf-managed library.
- Play, pause, seek, skip backward 15 seconds, and skip forward 30 seconds.
- Change playback speed and volume.
- Save and resume progress independently for each audiobook.
- Add or replace manual PNG, JPEG, and WebP covers.
- Keep all library data local with no account, cloud service, or telemetry added by ListenShelf.

## Windows downloads

- **Single-file portable EXE:** one file to copy and run. Its bundled .NET, Avalonia, SQLite, and LibVLC components are extracted to a private runtime cache when launched.
- **Portable ZIP:** extract the complete folder and run `ListenShelf.exe`.
- **Windows installer:** installs ListenShelf in Program Files and adds a Start menu shortcut.

All three builds are Windows x64 and self-contained; .NET and VLC do not need to be installed separately. ListenShelf currently stores library settings and progress under `%LocalAppData%\ListenShelf`, including when the executable itself is portable.

## Alpha warnings

- These builds are unsigned, so Windows may display an Unknown publisher or Microsoft Defender SmartScreen warning.
- Only DRM-free M4B playback is currently supported.
- Embedded covers, metadata, chapters, multi-file books, editing, and automatic updates are not implemented yet.
- Keep backups of your audiobooks. Managed imports make verified copies and linked imports do not modify source files, but this is still early software.

SHA-256 checksums are included in `SHA256SUMS.txt`.
